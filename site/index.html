simple-check

simple-check is an Clojure property-based testing tool inspired by QuickCheck. The core idea of simple-check (and QuickCheck) is that instead of enumerating expected input and output for unit tests, you write properties about your function that should hold true for all inputs. You can check out the project on Github.
